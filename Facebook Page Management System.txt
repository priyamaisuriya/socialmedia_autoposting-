Facebook Page Management System
Mini Project Overview Document
1. Project Title
Facebook Page Management System
2. Project Description

Facebook Page Management System ek web-based application che je multiple users ne potani Facebook Pages connect kari manage karva allow kare che.

Users:

Facebook login kari shake
Facebook Pages connect kari shake
Image/video/text posts publish kari shake
Comments joi shake
Multiple pages manage kari shake

System official Facebook Graph API no use karse.

3. Main Objective

Aa project no main goal:

Multiple Facebook users manage karva
Single dashboard mathi posts publish karva
Facebook Pages automate karva
Social media management easy banavvu
